🛠️ Reparar Schema de Base de Datos

Este script amplía los VARCHAR de la tabla usuarios para soportar nombres completos y cargos correctamente.

ALTER TABLE usuarios
  ALTER COLUMN nombre    TYPE VARCHAR(150),
  ALTER COLUMN tipo_doc  TYPE VARCHAR(5),
  ALTER COLUMN num_doc   TYPE VARCHAR(20),
  ALTER COLUMN cargo     TYPE VARCHAR(30),
  ALTER COLUMN area      TYPE VARCHAR(10);
